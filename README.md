# vue-h5

## 项目介绍
这是一个基于 vue-cli 的移动端项目汇总，持续更新中，主要功能如下：
- 通过 postcss 实现 vw/vh 自适应布局
- 通过 normalize.css 实现样式初始化
- 通过 pre-commit 校验实现图片压缩
- 全屏布局 Page 组件，可实现纵向布局自适应
- 常见移动端布局问题处理，主要为机型兼容问题
- 常见移动端开发实践：如 full-page，年度报告 H5
